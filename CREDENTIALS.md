## Creating Additional Users

Once logged in as admin:

1. Navigate to the **"Users"** tab
2. Click **"Add New User"**
3. Fill in the user details:
   - Username
   - Name
   - Password
   - Role (Admin, Sales Rep, or Driver)
4. For Driver accounts, link them to a driver profile for proper functionality

---

## User Roles

### Admin
- Full access to all features
- Manage products, drivers, users
- View all reports and dashboards
- Create and manage orders

### Sales Rep
- Create and manage orders
- View assigned orders
- Access sales reports
- View dashboard

### Driver
- View assigned orders (My Orders tab)
- View personal inventory (My Inventory tab)
- View earnings (My Earnings tab)
- Read-only access to order details

## Security Recommendations

1. **Change the default admin password immediately**
2. **Create individual user accounts** for each team member
3. **Use strong passwords** (minimum 8 characters, mix of letters/numbers/symbols)
4. **Disable the default admin account** once you've created your own admin user
5. **Keep this file secure** and delete it after setting up your users
